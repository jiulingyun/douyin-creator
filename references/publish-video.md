# 视频发布

> **状态：预留模块，待实现。** 以下为接口设计草案，具体参数和流程需通过实际调试确认。

## 概述

通过浏览器自动化在抖音创作者平台完成视频上传和发布，模拟真人操作。

## 命令

```bash
douyin-creator publish video --title "视频标题" --file "/path/to/video.mp4"
```

## 预留参数

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `--file` | 是 | - | 视频文件本地路径（MP4、MOV 等） |
| `--title` | 是 | - | 视频标题/描述 |
| `--topic` | 否 | - | 话题名称（不含 `#`） |
| `--cover` | 否 | - | 自定义封面图片路径 |
| `--location` | 否 | - | 添加位置信息 |
| `--visibility` | 否 | `public` | 可见性：`public`（公开）/ `friends`（好友可见）/ `private`（仅自己可见） |
| `--schedule` | 否 | - | 定时发布时间 |
| `--draft` | 否 | false | 存为草稿 |
| `--headless` | 否 | false | 无头模式运行 |

## 预期输出

```json
{
  "success": true,
  "action": "published",
  "title": "视频标题",
  "url": "https://..."
}
```
